# Очаги и Камины KZ — CLAUDE.md

## О проекте
Одностраничный сайт-каталог для компании по продаже костровых чаш, столов и печей в Казахстане.

**Технологии:** чистый HTML5, CSS и JavaScript — без фреймворков и сборщиков. Всё в одном файле `index.html`.

## Запуск локально

```bash
cd /Users/EfremShpakov/ochag-kz
python3 -m http.server 8080
```

Открыть в браузере: http://localhost:8080

> Прямое открытие `index.html` через `file://` тоже работает, но HTTP-сервер нужен для корректной загрузки шрифтов и изображений.

## Как вносить изменения

Всё находится в `index.html`:

- **Стили** — в теге `<style>` в `<head>`
- **Контент и товары** — в теге `<body>` (секции `#catalog`, `#contacts` и т.д.)
- **Логика** — в теге `<script>` в конце `<body>`

### Добавление/изменение товаров
Каждый товар — это `<div class="product-card" data-category="...">` в секции `#catalog`.
Категории: `bowl` (костровые чаши), `table` (столы), `oven` (печи).

### Изображения товаров
Положить фото в папку `images/` и добавить в карточку товара:
```html
<div class="product-img">
  <img src="images/название-файла.jpg" alt="Описание товара" loading="lazy">
</div>
```
